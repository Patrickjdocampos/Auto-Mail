#  Assistente de E-mail com IA

Um script em Python que automatiza a leitura e triagem de e-mails utilizando a API Generativa do Google (Gemini) para resumir e categorizar mensagens. Este é o primeiro projeto do meu portfólio de Engenharia de IA.

---

##  O Problema Resolvido

Este projeto combate a sobrecarga de informações na caixa de entrada do e-mail. Em vez de abrir manualmente cada mensagem, o assistente fornece resumos rápidos e inteligentes, permitindo que o usuário foque no que é realmente importante e economize tempo.

---

##  Tecnologias Utilizadas

* **Linguagem:** Python 3
* **Bibliotecas Principais:**
    * `imaplib`: Para conexão segura e leitura de e-mails via protocolo IMAP.
    * `dotenv`: Para gerenciamento seguro de credenciais e chaves de API.
    * `google-generativeai`: Para interação com a IA Gemini (será adicionado futuramente).
* **Ferramentas:** Git, GitHub, VS Code
